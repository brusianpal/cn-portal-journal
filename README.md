# cn-portal-journal

一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，仅作为 HTML 页面集合的存储与展示空间。

## 项目简介

本仓库旨在收集、整理并公开发布一系列独立的 HTML 页面。这些页面可能涵盖不同的主题、用途或时间节点，彼此之间没有强关联，仅共享“以 HTML 格式呈现”这一共同特征。

仓库本身不承担任何网站的运营或维护角色，仅作为静态页面的归档载体。欢迎浏览、参考或按需使用这些页面。

## 目录说明

```
cn-portal-journal/
├── index.html          # 仓库首页/导航页（可选）
├── pages/              # 独立 HTML 页面存放目录
│   ├── page-01.html
│   ├── page-02.html
│   └── ...
├── assets/             # 公共资源（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/`：所有独立 HTML 页面均存放于此，文件名通常反映页面内容或编号。
- `assets/`：页面可能使用的共享样式、脚本或图片资源，便于统一管理。
- `index.html`：如存在，可作为浏览所有页面的入口或目录页。

## 页面归档说明

- 每个 HTML 页面均为独立文件，可单独打开或引用。
- 页面内容可能来自不同时期、不同来源，归档时保持原始结构，仅做必要整理。
- 不保证所有页面均持续更新或维护，部分页面可能为历史存档。
- 页面内可能包含外部资源引用（如字体、库文件），但不保证这些外部资源永久可用。

## 贡献指南

1. Fork 本仓库。
2. 在 `pages/` 目录下添加新的 HTML 页面，或修改已有页面。
3. 确保页面独立可运行，且不包含恶意代码。
4. 提交 Pull Request，说明新增或修改的内容。
